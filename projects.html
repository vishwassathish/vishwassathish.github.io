Where do humans look when exploring for rewards?

An interactive visualization of human eye-tracking behavior during Atari gameplay, using the Atari-HEAD dataset. The project examines how visual attention relates to upcoming reward events across games with different reward structures — fast-paced (Breakout, Space Invaders) vs. sparse explorative (Montezuma’s Revenge). Built for CSE 512 Data Visualization, UW, Spring 2026.
